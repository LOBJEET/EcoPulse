# 🌿 EcoPulse - Quick Reference Card

## 🚀 Get Started in 30 Seconds

```bash
cd EcoPulseMobile
npm install
npm start
```

Then scan QR code with Expo Go (or press 'a' for Android)

## ⚡ Quick Troubleshooting

| Problem | Fix |
|---------|-----|
| Won't start | `npm install` then `npm start --clear` |
| Module errors | Restart terminal, try npm install again |
| Emulator issues | Restart Android Studio |
| Screen blank | Check imports in that screen file |
| Styles wrong | Verify colors.ts colors are imported |
